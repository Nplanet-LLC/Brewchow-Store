# Logo Setup Instructions

## Save the Logo Image

1. Save the logo image you provided (the one with black background) as:
   - `public/logo-dark.png`

2. The logo will automatically be used in:
   - Navbar (top navigation)
   - Footer
   - Favicon

## Current Logo Path
The code is looking for: `/logo-dark.png`

## Steps to Add Logo:
1. Right-click on the logo image in the chat
2. Save it to: `brewhaus-nextjs/public/logo-dark.png`
3. Refresh the browser - logo will appear automatically!

## Logo Specifications:
- Format: PNG with transparency
- Background: Black (#000000)
- Logo colors: Green tones (#2C4A3E, #7A9B8E)
- Size: Approximately 1024x1024px
